# AI Observability & RAG Performance Monitoring Dashboard

## Overview
This project builds an enterprise-grade AI observability system for monitoring RAG (Retrieval-Augmented Generation) pipelines.

It tracks system performance, retrieval quality, latency, and anomalies to ensure reliable AI operations.

---

## Key Features
- RAG pipeline performance monitoring
- Latency tracking and anomaly detection
- Retrieval quality scoring system
- AI system health scoring
- SLA monitoring by query type
- Drift and degradation detection

---

## Tech Stack
- Python (Pandas, NumPy)
- SQL (aggregation logic)
- Power BI (dashboard visualization)
- Time-series analysis

---

## Key Metrics
- Retrieval score accuracy
- Latency (avg & p95)
- Error rate
- System health score
- Degradation flags
- SLA violations

---

## Business Impact
- Improves AI system reliability
- Detects model degradation early
- Reduces downtime in AI pipelines
- Ensures SLA compliance
- Enables AI Ops monitoring

---

## How to Run
1. Open Google Colab
2. Run notebook step-by-step
3. Export CSV files
4. Load into Power BI dashboards

---

## Output Files
- rag_logs.csv
- rag_sla_metrics.csv
- rag_trend_analysis.csv

---

## Outcome
Transforms raw AI system logs into actionable observability insights for enterprise-scale RAG systems.
